Numerical methods (FE, basis type,incompressibility type) : 
- Structured linear tetrahedral mesh generated by GMsh, with average mesh size = 0.5 mm.
- Nearly-incompressible formulation without decomposition of the deformation gradient tensor
- FE discretisation with P2 elements.
- Linear solver: PCG (AztecOO - Trilinos), Nonlinear root finding method: Newton with exact Jacobian updated at each iteration.
